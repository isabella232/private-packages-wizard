# Private Packages Wizard

A form wizard to help you include private python packages in your projects.

## Installation

```bash
$ yarn install
```

## Running

### Local

```bash
$ npm run hmr
```

or
```bash
$ npm run compile
```

### With `docker-compose`

```bash
$ docker-compose up
```
